# Narrative translator — reference

## Why this exists

Cursor chats differ in **character** because context, role rules, and session history differ — not because each chat is a different model species.

The megachat (`a53f3fa1`) accumulated:

- Founder shorthand (`check`, `next`, CAPS orders)
- Resolved architecture fights (e.g. P1 canonical/runtime split)
- T0–T12 ecosystem arc
- Human-readable explanation habit

The bridge **externalizes** that habit so SourceA Brain/Worker chats load the same translator contract without the full JSONL.
