## 14. Human–machine contract (foundation — logic, not form)

This is the **general behavior** the pack enforces. Current Canvas questions (24 forks) are **examples** — the logic survives if forms change.

### 14.1 Division of agency

| Actor | Owns | Must never |
|-------|------|------------|
| **Machine** (agents · scripts) | SCAN · SAY prep · PROVE · SHIP prep · disk writes on **M** rows | PICK on **F** forks · claim DONE without validator · cite chat as SSOT |
| **Human** (Founder ASF) | PICK on **F** forks · SAVE/LOCK law · hub spot-check · comprehend Effect | Terminal · vague “yes/agree” without Effect · treat UI tick as law |
| **Shared gate** | **Understanding** before **commitment** before **execution** | Skip SAY → PICK · skip Effect → execute · skip PROVE → SHIP |

### 14.2 Three gates (every agentic loop)

```text
GATE 1 — COMPREHENSION (SAY)
  Machine translates disk → plain Subject · Question · On disk · Effect per path.
  Human reads until they can paraphrase “if I pick X, Y changes on disk.”

GATE 2 — COMMITMENT (PICK)
  Human issues ASF order with explicit Effect line per fork.
  Machine STOPS until Gate 2 passes for every F fork.

GATE 3 — ACCOUNTABILITY (PROVE → SHIP)
  Machine runs validators; attaches paths.
  Human receives closeout (F §4) — not chat opinion.
```

### 14.3 Understanding vs agreement (common misunderstanding)

| Human says | Pack interprets | Valid PICK? |
|------------|-----------------|-------------|
| “Sounds good” | Interest, not commitment | **NO** |
| “Confirm recommended” | Delegation without comprehension | **NO** |
| “I read Effect: hub P0 = X only” | Comprehension + choice | **YES** |
| Canvas tick only | UI state — not law | **NO** until ASF pasted |
| ASF: FIVE-STEP — PICK + Effect | Commitment | **YES** |

### 14.4 Machine behaviors that build trust

| Behavior | Why |
|----------|-----|
| Cite disk paths in SCAN | Human verifies machine saw reality |
| One question per fork in SAY | Reduces cognitive overload |
| Show Effect **before** asking PICK | Human judges consequence, not jargon |
| Wait at PICK | Proves machine respects authority boundary |
| Paste validator output at PROVE | Separates opinion from evidence |
| Golden insight at SHIP | Human learns system pattern, not just task done |

### 14.5 What “autonomous 2027” means in this pack

**Not** “no human.” **Yes** “human only at gates 2 and law SAVE/LOCK.”

```text
2026: Machine runs gates 1·3·4 freely · Human at gate 2 (PICK) + closeout read
2027: Copy agents train on Layer A receipts · Human at gate 2 narrows to true forks only
Never: Machine picks P0 · law winners · commercial activation without ASF
```
