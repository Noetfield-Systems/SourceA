## 2. Why hierarchies exist (philosophy — not bureaucracy)

Noise is fragmentation. Hierarchy exists so **one subject** has **one decision path**:

| Question | Without hierarchy | With P0–P7 |
|----------|-------------------|------------|
| What is true? | Chat, newest file, hub hero fight | **P0 SSOT** wins |
| Which file is law? | Grep 400 files | **P1 authority row** |
| What does agent obey? | Prompt memory | **P2 topic law** at path |
| How do we know it held? | “Trust me” | **P3 enforcer** FAIL |
| What does founder see? | Stale copy | **P4 projection** (RT or labeled LAG) |
| How do we run a session? | Ad-hoc | **P5 process machine** |
| Where do ideas wait? | Accidental law | **P6 staging only** |
| What do we reject? | Sprawl | **P7 noise** — archive · pointer |

**Why P0 is SSOT:** Only one apex can answer “what is our system?” — else every agent invents structure.  
**Why P7 exists:** Knowing what we **reject** is as important as what we keep — rejection **is** policy.
