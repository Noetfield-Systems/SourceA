# WitnessBC brand denylist — external policy

Deny patterns are **not** stored in this repository.

Install the Mac founder policy file:

`~/.sina/sourcea-brand-denylist-v1.json`

Or set `SOURCEA_BRAND_DENYLIST_PATH` in CI from a protected secret.

Schema: `witnessbc-brand-denylist-v1` with `deny_patterns_b64` and `allow_regex`.
