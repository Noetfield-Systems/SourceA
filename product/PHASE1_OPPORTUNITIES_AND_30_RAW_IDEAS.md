**Date:** 2026-06-04

## Part A — Opportunity identification

### Macro shifts (2026)

- AI generation is cheap; **verification, routing, and completion** are scarce.
- SMBs run many SaaS tools with **no execution layer** between them.
- PDFs, exports, and email attachments remain the **system of record** for millions of workflows.
- Vertical workers want **one job done** reliably, not a general assistant.
- Developers run agents locally without a standard **PASS/FAIL + artifact** habit.

### Opportunity zones (where demand is real)

| Zone | Core pain | Why tools fail today |
|------|-----------|----------------------|
| **Z1** | Prove work happened | Chat logs are not evidence |
| **Z2** | Move PDFs/email → action | OCR/chat read ≠ workflow |
| **Z3** | Repeat weekly ops without hiring | Zapier needs builders; PM tools don’t run |
| **Z4** | Creator/post pipeline after filming | Editing and repurposing still manual |
| **Z5** | Field job closeout | Photos + checklist + PDF sign-off fragmented |
| **Z6** | Compare/merge documents | Enterprise tools too heavy/expensive |
| **Z7** | Clean messy exports | Spreadsheets break; analysts waste hours |
| **Z8** | Niche first-pass drafts with citations | Generic LLM hallucinates on RFP/grants |
| **Z9** | Monitor external change (price, listing, reg) | Alerts without narrative |
| **Z10** | Small utilities on recurring micro-tasks | Spread across 5 free tools |

### Saturated (ideation should avoid center of gravity)

- Generic “AI chat for X”
- Undifferentiated blog/SEO writers
- Another AI image generator
- “All-in-one productivity” without execution

### Open bands (ideation should favor)

- Evidence + human approve before send
- PDF in → structured out → human confirm
- Vertical one-job agents (contractor, clinic intake, grant skim)
- Job-priced micro utilities (merge PDF, diff contract, categorize CSV)
- Scheduled narrative from monitors (not raw alerts)

## Part B — 30 raw product ideas

*Format: short name · problem · raw solution hook. No brand system, no stack, no pricing.*

---

### Z1 — Evidence / verification

1. **RunReceipt** — Agent runs are disputed · Auto log file + PASS/FAIL snapshot per run  
2. **ApproveSend** — Teams blast AI email without review · Hold outbound until one-tap approve  
3. **TaskProof** — Freelancers can’t prove delivery · Timestamped artifact bundle per completed task  
4. **ChatToClaim** — “AI did it” conflicts · Extract claims from transcript → checklist to verify  

### Z2 — Document intake / PDF

5. **InboxPDF** — PDFs trapped in email · Classify attachment → route to folder + task  
6. **MergePack** — Clients need one PDF from many · Drag-drop merge + page order + stamp  
7. **RedactLite** — SMB needs redaction before share · Auto-detect PII blocks → human confirm  
8. **FormToPDF** — Quotes/invoices rebuilt by hand · Form answers → branded PDF same day  
9. **SignPrep** — Wrong docs sent for signature · Validate required fields before e-sign link  

### Z3 — Business ops repetition

10. **WeekOps** — Owner repeats same Monday checks · Scriptable weekly run + short report  
11. **VendorOnboard** — Vendor setup is email chaos · Checklist runner + doc collection status  
12. **ChaseInvoice** — Unpaid invoices slip · Rules-based reminders from ledger export  
13. **StandupKill** — Standups exist to share status · Pull git/PM/ticket → one status page  
14. **HandoffNote** — Shift handoffs lose context · Structured handoff form → PDF for next shift  

### Z4 — Creator / content ops

15. **ClipMap** — Long video → shorts is tedious · Detect chapters → suggest clip ranges  
16. **CaptionGuard** — Wrong captions hurt brands · Style rules lint before publish  
17. **RepurposeQ** — One post should be 5 formats · Queue transformations with human order  
18. **ClientReel** — Agencies send files in chaos · Branded delivery page per client per month  

### Z5 — Field / vertical completion

19. **SiteClose** — Field visits lack proof · Photo + checklist → completion PDF  
20. **PermitStack** — Contractors lose permit docs · Assemble jurisdiction packet from template set  
21. **IntakeSum** — Clinics drown in intake forms · Summarize intake → draft note for clinician edit  
22. **RFP skim** — RFPs too long to bid · Extract requirements table + due dates + risks  

### Z6 — Document compare / intelligence

23. **ClauseDiff** — Contract changes unreadable · Side-by-side clause-level diff for non-lawyers  
24. **PolicyMatch** — “Does this violate our policy?” manual · Rule list check against doc text  
25. **SpecExtract** — Specs buried in PDF · Pull requirements into structured table  

### Z7 — Data extract / monitor

26. **CSVDoctor** — Messy exports break models · Profile + fix suggestions + validation report  
27. **ShelfWatch** — Competitor prices change silently · Diff catalog scrape → weekly memo  
28. **ReceiptSort** — Shoebox receipts · Photo/PDF → categories with confirm batch  

### Z8 — Dev / technical micro

29. **TranscriptYAML** — Chat outcomes not machine-readable · Parse agent transcript → strict YAML report  
30. **SmokeRemote** — “Did deploy work?” from phone · Trigger allowed smoke script → return PASS/FAIL only
