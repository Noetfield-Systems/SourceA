Lifecycle Architecture

The full lifecycle. Every agent. Every action.

01

Agent identity established

Cryptographic birth certificate issued. Archetype assigned. Behavioral baseline set.

runtime.birth(agent_id, scope, archetype)
02

Agent proposes action

Any framework. LangGraph, CrewAI, AutoGen, or custom. Framework-agnostic.

govern_action(action, context)
03

Behavioral Control Plane™ evaluates

20 dimensions scored simultaneously. Veto checks, trust history, context profiles.

Verdict issued

ALLOW / DENY / ESCALATE / MODIFY / SUSPEND. The commit boundary is the evaluation itself.
