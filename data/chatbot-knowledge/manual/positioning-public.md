# SourceA positioning (public)

## One line

SourceA is an AI execution platform powered by Forge — real builds, automations, and agent workflows, with a verifiable receipt on every run.

## What SourceA is NOT

- Not "just records" or verification-only software
- Not a generic ChatGPT wrapper
- Not a pitch that opens with dollar amounts

## What Forge is

Forge Terminal is the execution desk — living chat, workspace, agents, quality gate, cloud dispatch.

Public try (no install): `/sourcea/forge/terminal`

Flow: idea → prompt forge → agents → quality gate → execute (Cursor / cloud)
