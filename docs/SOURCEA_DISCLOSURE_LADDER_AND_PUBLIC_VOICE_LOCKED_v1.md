## 2. Five-tier disclosure ladder (T0–T4)

```text
T0  PUBLIC WEB       → fear + outcome + one CTA + sanitized proof
T1  ICP BUYER        → brand SKU + live demo + receipt concept
T2  SERIOUS PILOT    → governance process summary (no scripts)
T3  NDA DILIGENCE    → portfolio OS + moat categories + eval methodology
T4  FOUNDER / AGENTS → full disk truth (~/.sina · brain · gates · rooms)
```

### T0 — Public web · LinkedIn top-of-funnel · anonymous prospect

**Audience:** Website visitor · LinkedIn scroll · cold awareness  

**Show:**
- Customer job and fear (Vanta/Credo pattern — continuous proof, not features)
- One-line: *Prove every agent action — before the model runs, after it ships*
- Category: *Agentic trust infrastructure*
- Product UI or commercial film (tier A/B hero — **0% face** on GRC lanes per film routing)
- High-level loop only:

```text
Intent → Gate → Execute → Receipt → Proof
```

- Honest limits when live eval blocked (structural mode — golden report L75–76)
- Perimeter from `linkedin-voice.yaml`: pre-execution advisory · no custody · no payment initiation

**Hide:**
- Brain · session gate · SASCIP · `sa-*` queue · investigator/judge/Auto Runtime specialist rooms
- Mac control plane · FBE internals · campus node counts
- Five-entity portfolio map · RunReceipt factory · Goal 1 sa-step progress
- MSB/TrustField as **homepage hero** (brand pages OK on own domains)
- Internal agent names · hub inject · `~/.sina` paths · LOCKED law filenames in marketing

**Primary CTA:** one only (design partner · demo · contact) — Linear/Vercel pattern

**SSOT surfaces:** SourceA landing · brand sites · LinkedIn · `linkedin-voice.yaml`

### T1 — ICP buyer · W3 commercial · named account outbound

**Audience:** Copilot buyer · RWA/MSB desk · agency buyer · design partner  

**Show:** brand-specific outcome · live BLOCK/replay · pilot economics · one metric  

**One-sentence table** (`FAST_BUSINESS_MODEL v2` §10 — verbatim):

| Audience | Say |
|----------|-----|
| Copilot buyer | Your agents act alone. Noetfield enforces your rules before they act — and proves it after. |
| RWA/MSB buyer | You need evidence examiners can replay — not another tokenization pitch. |
| Agency buyer | Fifteen minutes of live BLOCK and receipt closes deposits. |
| Investor (later) | Pre-LLM governance runtime with deployable factory apps. |
| Founder | Mac manages business. Cloud runs factories. One deposit proves the stack. |

**Hide:**
- SourceA spine as **hero** — footer only: *Powered by SourceA governed execution — advisory only · no custody · no payment initiation*
- ICP compile machinery · machine loop scores · RRL stage names in buyer-facing body

**Ship gate:** `sina_read_score_pct` ≥ 90 — `FACTORY_VOCABULARY…` §4  

**Machine:** ICP compile via `scripts/icp_output_compiler_v1.py` · film via `data/commercial-film-routing-v1.json`

**Vocabulary:** say **ICP compile** — never “forge email”; **Forge** = product only (`~/Desktop/forge`)

- Investigator/judge room receipts · SASCIP probe details · sa-queue contents
