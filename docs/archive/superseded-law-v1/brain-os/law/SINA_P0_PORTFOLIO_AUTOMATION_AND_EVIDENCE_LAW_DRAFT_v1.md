### PILLAR A — Agentic social & surface operations (24/7)

**Definition:** All public and semi-public surfaces — websites, landing pages, LinkedIn, X, company pages, hub panels, outreach inboxes — are **operated by controlled agents**, not manual founder sessions.

| Rule | Requirement |
|------|-------------|
| A1 | **No single-page strategy** — LinkedIn alone is insufficient |
| A2 | **Control plane** — Sina Command + hub panels are operator UI; agents execute |
| A3 | **24/7 loop** — scheduled + event-driven runs (n8n/Make-class triggers) |
| A4 | **Receipt per action** — post sent · email queued · page updated → spine row |
| A5 | **Validator before publish** — governance gate on outbound content |
| A6 | **Founder** — approve bounded orders only (Hub tap); never manual post/email/call |
| A7 | **SEMEJ / Chrome** — optional execution lane; not SSOT |

| Function | Precedent companies | Our implementation |
|----------|---------------------|-------------------|
| Multi-channel scheduling | Buffer · Hootsuite · Sprout Social | n8n + agent content queue |
| Workflow glue | **n8n** · **Make** · **Zapier** | `SINA_AUTOMATION_SPINE_AND_N8N` T4b |

| CMS / site updates | Webflow · WordPress + Zapier | Portfolio repos + automation hook |

### PILLAR B — Commercial automation only (evidence law)

**Definition:** No feature, integration, or “smart idea” enters the build queue unless it maps to a **named commercial precedent** and a **receipt-class automation pattern**.

| Rule | Requirement |
|------|-------------|
| B1 | **No trial-and-error stage** — rejected at intake |
| B2 | **Evidence row required** — company · product · pattern · link |
| B3 | **Commercial work definition** — methods used by revenue-stage automation vendors (n8n, Make, Zapier, HubSpot, Apollo, etc.) |
| B4 | **Non-commercial work** — research essays, OS naming, paradigm pivots without W1/W2/W3 gate → **DELETE** |
| B5 | **Rejection default** — no evidence → no speech in hub, chat, or roadmap |
| B6 | **EXTERNAL_CRITIC** — still report-only; never satisfies B2 alone |
