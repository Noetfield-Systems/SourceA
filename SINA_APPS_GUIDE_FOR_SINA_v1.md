# Sina apps — teacher guide (for Sina)

**Best experience:** open Sina Command → **Guide** tab. Lessons work like tips when you install a new app — tap **Try it now** on each step.

This file is a backup; the live teacher walkthrough is in the app.

---

## In the app (recommended)

1. Double-click **Sina Command** on Desktop.
2. Sidebar → **Guide**.
3. Start **Lesson 1** (opens automatically) and work down the list.
4. Use the chip shortcuts at the top (Home, Apps, Notes, AI Coach, …).

---

## One-minute start

| Step | Do this |
|------|---------|
| 1 | Open via Desktop **Sina Command** — pill must say **Live** |
| 2 | **Home** → pick a page (Today, Actions, Recent work) — one screen per topic |
| 2b | **Actions** — dispatch, ingest, brief (one tap each) |
| 2c | **Next steps** — optional disk queue glance on Worker Hub (execution = RUN INBOX in Worker chat) |
| 3 | **Refresh** (gold, top right) after real work |
| 4 | **Your notes** — write fixes for Cursor without screenshots |

**Never use Terminal** for daily work — every order has a button in the app.
