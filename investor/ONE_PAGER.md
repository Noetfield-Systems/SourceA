# One pager — Sina portfolio operating system

**ASF · Multi-business founder platform · June 2026**

---

## Problem

One founder runs **several operating companies** (infrastructure, platforms, foundations, future products). Traditional setup = founder bottleneck: work only happens when sitting at one Mac, in five separate chats, with no single view of “what shipped today.”

## Solution

**Portfolio operating system:** one Mac spine + phone remote control + per-company execution lanes → each business moves toward **live product and revenue** on a dated roadmap.

## What is already real (not slideware)

- **Remote operations:** Start the same production pipeline from an iPhone on **cellular** (Tailscale), not only home Wi‑Fi — Mac runs architect → dispatch → ingest → report.
- **Multi-repo governance:** Five businesses have ranked daily priorities, execution logs, and plan files — not ad-hoc folders.
- **Separation of concerns:** Internal orchestra (private) vs **commercial App Store app** (separate program, separate deck).
