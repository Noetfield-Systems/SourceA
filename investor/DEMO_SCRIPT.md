# Live demo script — 5 minutes

**Goal:** End on **customer result**, not PASS/FAIL.

---

## Before the meeting

| Check | |
|-------|---|
| Mac on, `npm start` running in DevBridge | |
| Tailscale connected Mac + demo phone | |
| One **business** artifact ready (VIRLUX staging URL, TrustField screenshot, etc.) | |
| Wi‑Fi **off** on phone optional (proves away-from-home) | |

---

## Minute 0–1 — Frame (talk)

> “I’m not demoing a university test. I’m showing how I **run several companies** when I’m not at my desk. The money is what each company ships — I’ll end on that.”

---

## Minute 1–2 — Remote ops (show)

1. iPhone Safari → `http://100.85.10.79:3004/...&lane=full_m8` (bookmark).
2. Connect → RUN SYSTEM → log scrolls (real pipeline).
3. One line: “This used to require me sitting here. Now I can **start** it from the car.”

**Do not** celebrate PASS. Say: “Mac is running today’s ops; the **deliverable** is what lands in each repo.”

---

## Minute 2–3 — Handoff (honest, 20 seconds)

> “Work orders are generated on the Mac. Auto-paste into every Cursor chat is the next ops fix — today it can be one company from the Agent screen. The bottleneck is handoff, not connectivity.”

---

## Minute 3–5 — Customer result (pivot — **this is the demo**)

Open laptop or second tab:

| If leading with | Show |
|-----------------|------|
| VIRLUX | Staging URL, product, checkout path |
| TrustField | Demo login or workflow |
| Mono | Bot approval or API health |
| 777 | Public site + outreach plan |
| Cursor OS Pro *(only if App Store meeting)* | Separate app — pair + tabs |

Close:

> “Funding accelerates **these** milestones across the portfolio, not more internal tests.”

---

## If something breaks

| Failure | Say |
|---------|-----|
| Phone won’t load | “Mac asleep or bridge stopped — capability is proven; here’s recording / second device.” |
| RUN SYSTEM FAIL | “Ops bug on Mac — here’s yesterday’s PASS and today’s business artifact.” |
| No staging URL yet | “Honest stage: remote ops done; VIRLUX staging is the 30-day milestone — here’s ROADMAP.” |

---

## Never say in the room

- smoke test  
- g3_tailscale pass  
- ingest YAML  
- plan.json marked_done  

---

## Leave-behind

Email `ONE_PAGER.md` + `ROADMAP.md` — not terminal logs.
