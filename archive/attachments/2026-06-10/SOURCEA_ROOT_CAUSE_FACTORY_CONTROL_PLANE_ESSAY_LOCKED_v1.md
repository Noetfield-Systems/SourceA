### Invariant 1 — ONE mode (machine)

```text
FREEZE | AUDIT | SINGLE_SA | SHIP_MAINTAINER
```

| Mode | Allowed | Forbidden |
|------|---------|-----------|
| **FREEZE** (default) | stop · audit · hub refresh | autodrain · pack_loop · batch closeout |
| **AUDIT** | read-only disk · tables | any spawn drain |
| **SINGLE_SA** | one INBOX turn · max 1 pack receipt | max-turns > 1 without ASF token |
| **SHIP_MAINTAINER** | bounded paths + EDIT ALLOWED | drain · REGISTRY bulk |

Hub + ACTIVE_NOW display mode. Plan todos **void** when mode ≠ SINGLE_SA.

### Invariant 2 — ONE now-line (machine)

**All** chat progress · monitor header · inject · Brain status cite **only this**. Kills 013·014·dual narrative class.

- `pack-NN.json` or receipt path in chat

**Miss → auto FREEZE** (touch flag + stop receipt). Kills “stuck” feeling.

### Invariant 4 — ONE spawn gate
