## 2. Cost tiers (monthly stack)

### Tier 0 — Bootstrap ($0–15/mo)

| Tool | Cost | Role |
|------|------|------|
| OBS Studio | Free | Capture |
| QuickTime | Free | Backup capture |
| YouTube unlisted | Free | Host |
| iMovie / Clipchamp | Free | Basic trim |

**Output quality:** 7/10 polish · 10/10 truth  
**Best for:** Draft rehearsals until validator PASS  
**Avoid:** Shipping W1 to investors on Tier 0 alone

### Tier 1 — Recommended ($33–55/mo) ← **DEFAULT**

| Tool | Cost | Role |
|------|------|------|
| Screen Studio | $9/mo annual or ~$29 mo | **Capture + auto-zoom** |
| Descript Creator | $24/mo | **Edit + captions** |
| YouTube/Vimeo free tier | $0 | Host |

**Output quality:** 9/10 · matches Temporal/Stripe demo bar  
**Annual:** ~$396 (Screen Studio one-time ~$89 also viable)

### Tier 2 — Scale ($80–120/mo)

Add only when **NW1 pipeline active** and need volume:

| Tool | Cost | When to add |
|------|------|-------------|
| Clueso or Trupeer | $49–120 | Need video + SOP docs from same capture |
| Wistia | $19+ | Analytics on demo views for sales |
| Record Once | $29+ | Frequent UI changes, hate re-recording |

**Do not add** until first deposit or 10+ outbound demos sent.

### Tier 3 — Waste for pre-W3 ($150+/mo)

| Tool | Why skip now |
|------|--------------|
| HeyGen / Synthesia | Avatar ≠ terminal proof |
| Demosmith autonomous | Wrong for live CLI |
| Runway subscription | B-roll not needed for W1 |
| Agency ($2K–10K/video) | ROI negative before W3 |
| Camtasia + Premiere stack | Time cost > benefit solo founder |

## 4. Cost intelligence rules (LOCKED for ops)

### Rule 1 — One master, many derivatives
Record V-W1 once. Export:
- 5-min investor version  
- 90s Buyer 1 version  
- 60s Noetfield hook (compliance voiceover swap in Descript)  
- GIF beats for outreach  

**Savings:** 75% vs four separate productions.

### Rule 2 — Pay for polish, not generation
Generative video (Runway, Pika) costs **$0.50–2 per second** of footage with no proof value.  
Screen polish (Screen Studio) costs **~$0.10 per minute** of finished demo.

### Rule 3 — Category match before subscribe
| Need | Pay for | Never pay for |
|------|---------|---------------|
| Live CLI proof | Screen Studio, OBS, Descript | HeyGen, Synthesia |
| Compliance buyer | Same V-W1 + NF voice track | Separate avatar studio |
| Sales embed post-W3 | Supademo free → paid | Full Demosmith until UI stable |
| Internal async | Loom free tier | Camtasia |

### Rule 4 — Time is the hidden cost
| Approach | Founder hours | $ opportunity cost @ $200/h |
|----------|---------------|----------------------------|
| Screen Studio + Descript | 6 h | $1,200 |
| Camtasia manual | 15 h | $3,000 |
| Agency | 2 h review | $400 + $5K cash |

**Cheapest cash ≠ cheapest total cost.**

### Rule 5 — Refresh trigger only
Re-film V-W1 only when:
- Validator demo script changes beats  
- BLOCK/ALLOW/tamper behavior changes  
- Export bundle UI ships  

Not when: copy tweak, new landing page, advisor aesthetic opinion.

### Rule 6 — 15% energy cap (SSOT v3.1 §12)
Demo/video work ≤ **15% weekly energy**. If tool learning exceeds 15%, **stop** — use Tier 1 defaults only.

## 9. Golden cost rules (10)

1. **Film once · cut thrice** — master V-W1 drives all layers.

8. **Track hours not just SaaS** — 15% energy law.  
9. **First money target ($2K) >> video perfection.**  
10. **Proof beats polish** — tamper FAIL visible > 4K intro animation.
