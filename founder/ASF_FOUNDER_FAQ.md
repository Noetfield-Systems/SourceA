## “When does the prompt refresh?”

**When maintainer runs SinaPromptOS dispatch-day** (or future **Actions** button in hub).

Typing **IMPLEMENT** alone does **not** refresh. Use **Repos** → **Copy lane brief** for the latest paste file.

## “Paste or just say IMPLEMENT?”

**Paste the lane brief** (from hub **Repos**). IMPLEMENT alone only works in the **same** chat if context is already there.
